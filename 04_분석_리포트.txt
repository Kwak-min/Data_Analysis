4. 모델별 특징 및 분석
--------------------------------------------------------------------------------

[K-최근접 이웃(KNN) 회귀]
- 거리 기반 비모수적 머신러닝 기법
- 과거 유사 패턴을 찾아 예측 수행
- 특징:
  * 직관적이고 이해하기 쉬운 알고리즘
  * 학습 시간이 짧고 구현이 간단함
  * 거리 기반이므로 정규화(스케일링) 필수
  * K값 선택이 성능에 큰 영향
- 장점: 비선형 패턴 포착, 빠른 학습
- 단점: 예측 시간이 상대적으로 느림, 차원의 저주

[RNN (Recurrent Neural Network - 순환 신경망)]
- 시퀀스 데이터 처리에 특화된 딥러닝 기법
- 이전 시점의 정보를 순환적으로 전달하여 시계열 패턴 학습
- 특징:
  * 시간적 의존성(temporal dependency) 학습 가능
  * 가변 길이 시퀀스 처리 가능
  * 은닉 상태(hidden state)로 과거 정보 기억
  * tanh 활성화 함수로 비선형성 확보
- 장점: 복잡한 시계열 패턴 포착, 자동 피처 추출
- 단점: 학습 시간이 길고, 하이퍼파라미터 튜닝 필요
- 구조: 3층 RNN (128→64→32 유닛) + Dropout + Dense 레이어

--------------------------------------------------------------------------------
5. 머신러닝 vs 딥러닝 비교 분석
--------------------------------------------------------------------------------

[머신러닝: KNN]
✓ 학습 속도: 매우 빠름
✓ 해석 가능성: 높음 (최근접 이웃 확인 가능)
✓ 데이터 요구량: 상대적으로 적음
✓ 적용 난이도: 쉬움

[딥러닝: RNN]
✓ 학습 속도: 느림 (에폭 반복 필요)
✓ 해석 가능성: 낮음 (블랙박스 모델)
✓ 데이터 요구량: 많음
✓ 복잡도: 높음 (신경망 구조 설계 필요)
✓ 성능: 복잡한 패턴에서 우수
